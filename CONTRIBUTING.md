# Hướng Dẫn Đóng Góp Cho Sinh Viên

1. Tạo branch riêng cho bài làm.
2. Commit nhỏ, message rõ ràng.
3. Không commit `.env` hoặc API key.
4. Chạy `pytest` trước khi nộp.
5. Nếu có cài đầy đủ công cụ dev, chạy thêm `make lint` và `make typecheck`.
6. Viết hoặc cập nhật báo cáo trong `reports/benchmark_report.md`.
7. Đảm bảo report có đủ phần `Analysis` và `Trace`.
